## ‚úÖ Page: Booking Confirmation

### Description

This page confirms all booking details, including selected flight, passenger info, and payment summary.

---

### üè∑Ô∏è Legacy Endpoint Reference (Split into Two Clear Calls)

---

#### 1Ô∏è‚É£ Booking Confirmation Endpoint

- **Purpose:**  
  Called once the confirmation page is entered to retrieve the booking confirmation view.

- **Request:**  
  `GET odeysysadmin/flight/bookingConfirmation?isCoreSystem=true`

- **Query Parameters:**

  - `isCoreSystem` (boolean) ‚Äì Indicates if the request originates from a core system.

- **Response:**  
  `200 OK` ‚Äì Returns a **ModelAndView** (HTML content rendered on server side)

---

#### 2Ô∏è‚É£ Make Payment Endpoint

- **Purpose:**  
  Processes the booking payment and redirects to the final status page.

- **Request:**  
  `POST odeysysadmin/flight/processMakePayment?idempotency-key=3d536bf3-5b40-4c99-8e99-ed4796357d20`

- **Query Parameters:**

  - `idempotency-key` (string) ‚Äì Unique key to prevent duplicate transactions.

- **Response:**  
  `302 Found` ‚Äì Redirects after payment processing is completed.

---

### üè∑Ô∏è Legacy Structure

- **Format:** Verbose object with many unused fields and nulls
- **Delivery:** Injected in `ModelAndView` and used in JSP rendering
